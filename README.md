# enhancing-image-sharpness
Python implementation of the paper "Enhancing Image Sharpness by Modified Unsharp Masking Using Coefficient Bounds Obtained for a Subclass of Analytic Functions".  Paper link: https://www.nature.com/articles/s41598-023-45938-y . This project is part of our "Image Processing - CSE829" course assignment in IIT, DU.
